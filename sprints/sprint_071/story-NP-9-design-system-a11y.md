# Story NP-9 — Design System & Accessibility

**Status:** DONE
**Priority:** P0

## Acceptance Criteria

- [x] All colors, spacing, and typography use CSS custom properties from the design system
- [x] Smooth transitions on toggle state changes, card hover, and save bar appearance
- [x] `prefers-reduced-motion: reduce` media query disables all animations
- [x] All interactive elements are keyboard navigable with visible focus indicators
- [x] Toggle switches have associated `aria-label` or `aria-labelledby` attributes
- [x] Color contrast meets WCAG 2.1 AA minimum ratio (4.5:1 text, 3:1 UI components)
- [x] Screen reader announcements for toggle state changes and save confirmation
