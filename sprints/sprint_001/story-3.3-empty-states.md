# Story 3.3 — Empty States with CTAs

**Epic:** Loading States & Error Handling
**Priority:** HIGH
**Status:** DONE

## User Story
As a new user who just signed up,
I want helpful guidance on empty screens,
so that I know what to do next instead of seeing a blank page.

## Scope
- Driver tickets list: "No tickets yet — submit your first one" + "Submit Ticket" button
- Driver messages: "No conversations yet — your attorney will message you here"
- Driver notifications: "You're all caught up!" with checkmark icon
- Carrier dashboard (no drivers): "Add your first driver to get started" + "Add Driver" button

## Acceptance Criteria
- [ ] Each empty state shows a clear message and (where applicable) a primary CTA
- [ ] CTA buttons navigate to the correct action screen
- [ ] Empty states do not appear while loading (skeleton shows during load, empty state after)
- [ ] 44x44px touch targets on CTA buttons
